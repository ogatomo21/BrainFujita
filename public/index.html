自己肯定感爆上げ↑↑プログラミング言語

BrainFujitaは、Brainf*ck派生言語であり、学園アイドルマスターに登場する藤田ことねのソロ曲に出現する歌詞及びことぬい言語(ナー語)をコマンドとして使用することで、頭を空っぽにしてコーディングすることができる世界一？可愛いプログラミング言語です。

変換: BrainFujita → Brainf*ck
実行: 標準出力・バイト列・メモリ
UTF-8 拡張モード対応
Playground を試す README を開く

命令セット

  • しゅき / だいしゅき — ポインタ移動
  • ナー / セナー — セル増減
  • せーの / ことね — 出力 / 入力
  • ナ？ / セナ？ — ループ

UTF-8 拡張モード

BrainFujitaでは、UTF-8拡張モードを独自実装しており、ソース先頭に #!自己肯定感爆上げちゅ～ または #!自己肯定感爆上げ中 を追加することで UTF-8 バイト単位の入出力モードに切り替わります。
