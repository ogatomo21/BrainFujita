# BrainFujita - 自己肯定感爆上げ↑↑プログラミング言語

## 概要

BrainFujitaは、BrainF*ck派生言語であり、学園アイドルマスターに登場する藤田ことねのソロ曲に出現する歌詞及びことぬい言語(ナー語)をコマンドとして使用することで、頭を空っぽにしてコーディングすることができる世界一？可愛いプログラミング言語です。

## 仕様

- プログラム拡張子 `.kf` (Kotone Fujita Program File)
- インタープリター(`.kf` の実行) `fki` (Kotone Fujita Interpreter)
- コンバーター(`.kf` `.bf` 間の相互変換) `fkc` (Kotone Fujita Converter)

プログラム内では、ひらがなや全角・半角カタカナ、全角伸ばし棒と半角ハイフン、全角疑問符と半角クエスチョンマークは同一視されます。命令セット以外の文字はコメントとして無視されます。例えば、`せなー` と `セナー` と `ｾﾅ-` はすべて同じコマンドとみなされます。

### 命令セット

| コマンド | BF相当 | 説明( [Wikipedia](https://ja.wikipedia.org/wiki/Brainfuck) より) |
| --- | --- | --- |
| `しゅき` | > | ポインタをインクリメントする |
| `だいしゅき` | < | ポインタをデクリメントする |
| `ナー` | + | ポインタが指す値をインクリメントする |
| `セナー` | - | ポインタが指す値をデクリメントする |
| `せーの` | . | ポインタが指す値を出力に書き出す |
| `ことね` | , | 入力から1バイト読み込んで、ポインタが指す先に代入する |
| `ナ？` | [ | ポインタが指す値が0なら、対応する ` ] ` の直後にジャンプする |
| `セナ？` | ] | ポインタが指す値が0でないなら、対応する ` [ ` の直後にジャンプする |

### BrainFujita独自拡張

ファイルの1行目が `#!自己肯定感爆上げちゅ～↑↑` または `#!自己肯定感爆上げ中` の場合のみ、BrainFuck 独自の UTF-8 拡張モードで実行します。
このモードでは、入力と出力を UTF-8 エンコードされたバイト列として扱います。`ことね (BF: ,)` と `せーの (BF: .)` は 1 命令で 1 バイトを読み書きし、テープのセルも 8bit です。日本語などのマルチバイト文字は、UTF-8 の複数バイト列として処理されます。

```
あ → E3 81 82 0A 
い → E3 81 84 0A
う → E3 81 86 0A
え → E3 81 88 0A
お → E3 81 8A 0A
```

## インタープリター - fki (Fujita Kotone Interpreter)

fkiは、BrainFujitaプログラムを実行するコマンドラインツールです。

### 実行

```text
fki.exe input.kf
```

```text
fki.exe --version
```

構文エラーがある場合は、行・列つきでエラー内容を表示して終了します。

## コンバーター - fkc (Fujita Kotone Converter)

fkcは、BrainFujitaとBrainF*ckの相互変換を行うコマンドラインツールです。

### 変換

```text
fkc.exe convert input.kf (output.bf)
```

### 逆変換

```text
fkc.exe revert input.bf (output.kf)
```

出力先を省略した場合は、入力ファイル名の拡張子だけを `.bf` または `.kf` に置き換えます。

構文エラーがある場合は、行・列つきでエラー内容を表示して終了します。

## ビルド

```text
make build
```

`make build` で `fkc.exe` と `fki.exe` をまとめて作成します。
