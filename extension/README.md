# BrainFujita Extension

English follows Japanese.

## 概要
BrainFujita Extensionは、学園アイドルマスターに登場する藤田ことねから着想を得て開発されたBrainf*ck派生言語「BrainFujita」のVisual Studio Code拡張機能です。BrainFujitaの構文のハイライトや、フォーマットの提供を行います。

## Overview
BrainFujita Extension is a Visual Studio Code extension for BrainFujita, a Brainf*ck derivative language inspired by Kotone Fujita from Gakuen Idolmaster. It provides syntax highlighting and formatting for BrainFujita.
